Приветствую! Меня зовут Денис, я из Норильска. Последние два года я самостоятельно изучаю веб-разработку. Мне интересен процесс создания сайтов, и я планирую развиваться в этой профессии. За время обучения я освоил верстку многостраничных сайтов и интернет-магазинов, практикуясь на реальных макетах из Figma. Мой стек включает HTML5/CSS3, БЭМ-нейминг, препроцессоры SCSS и сборщик Gulp. Большое внимание я уделяю качественной адаптивной верстке. Также я внедряю базовый JavaScript для оживления интерфейсов и стараюсь аккуратно выдерживать точность макета. Моя цель сейчас — найти команду, стажировку или первые практические задачи, где я смогу применить свои знания, принести пользу и продолжить расти как специалист.

Навыки

Тип: Корпоративный многостраничный сайт

Описание: Адаптивная вёрстка многостраничного сайта для логистической компании. Реализована удобная структура для навигации по направлениям перевозок, тарифные сетки и формы для расчета стоимости доставки.

Стек: HTML5, CSS3 (Flexbox/Grid), JS, БЭМ, SCSS, Gulp.

Тип: Промо-страница (Лендинг)

Описание: Коммерческий одностраничный сайт с упором на высокую скорость загрузки и адаптив под мобильные устройства.

Стек: HTML5, CSS3, JS, БЭМ, SCSS, Gulp, PixelPerfect, Яндекс API

Тип: Корпоративный многостраничный сайт

Описание: Многостраничный сайт-каталог для мебельного производства. Свёрстаны страницы категорий, карточки индивидуальных проектов и формы для отправки эскизов на замер. Особое внимание уделено оптимизации тяжелых изображений мебели.

Стек: HTML5, CSS3, SCSS, БЭМ, Gulp, оптимизация графики.

Тип: Интернет-магазин

Описание: масштабная главная страница интернет-магазина с большим количеством разнородных контентных блоков и товарных сеток. Комплексное навигационное меню со сложной структурой категорий. Адаптивная сетка карточек товаров.

Стек: HTML5, CSS3 (Grid Layout / Flexbox), БЭМ, SCSS, JavaScript, Gulp.

Тип: E-commerce проект (Интернет-магазин)

Описание: Современный интернет-магазин аудиосистем для автомобилей. Свёрстана сложная структура каталога с фильтрами, карточки товаров с выбором модификаций и интерфейс корзины для оформления заказа.

Стек: HTML5, CSS3 (Grid Layout), SCSS, БЭМ, JavaScript, Gulp.
